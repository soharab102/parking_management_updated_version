# Parking Management System

A desktop-based Parking Management System developed using C# WinForms and SQL Server.

## Features

* Vehicle Entry Management
* Vehicle Exit Management
* Parking Payment System
* Admin Panel
* User Registration & Login
* Dashboard Overview
* Update/Delete Parking Records
* SQL Server Database Integration

## Technologies Used

* C#
* WinForms
* SQL Server
* ADO.NET
* Visual Studio

## Setup Instructions

### 1. Clone Repository

```bash
git clone https://github.com/soharab102/parking_management_updated_version.git
```

### 2. Open Project

Open the `.sln` file using Visual Studio.

### 3. Restore Database

Restore the provided `.bak` database file using SQL Server Management Studio (SSMS).

### 4. Update Connection String

Update the SQL connection string according to your SQL Server instance.

Example:

```csharp
Data Source=SOHARAB\SQLEXPRESS;Initial Catalog=ParkingManagementV2;Integrated Security=True;TrustServerCertificate=True;
```

### 5. Run Project

* Build Solution (`Ctrl + Shift + B`)
* Run (`F5`)
